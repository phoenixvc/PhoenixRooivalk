# Shared Parts

Parts that appear across many builds (e.g. Phase-1):

| Part | Typical use |
|------|-------------|
| ESP32 (DevKitC / WROOM-32, etc.) | Actuator controller, optional ESP32-CAM node |
| MP1584EN (or similar 3A buck) | 5V/6V rail for servos and logic |
| Jumper wires | Breadboard and module connections |
| Buzzer | Local alarm |
| LED | Status and alarm indication |

Quantities and exact variants depend on the product and option (1a/1b/1c). See each [product doc](../phase1/products/index.mdx) for BOMs.
