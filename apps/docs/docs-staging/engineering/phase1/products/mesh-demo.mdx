# SkyWatch Mesh Demo

Mesh-capable detection demo. Multiple nodes; same common modules (power, alarms, networking, compute).

## Comparison (1a / 1b / 1c)

| Option | Cost | Best for |
|--------|------|----------|
| 1a | Lowest | Minimal mesh |
| 1b | Medium | Default mesh demo |
| 1c | Higher | Robust mesh |

## BOM (outline)

Per node: compute, camera, power, networking. See [common modules](../../common/index.mdx).

### Parts list (orderable)

| Part | Qty | Notes (1a: 2 nodes) |
|------|-----|---------------------|
| Raspberry Pi 4 (2GB) | 2 | One per node |
| Pi Camera v3 or USB webcam | 2 | One per node |
| 1GbE switch | 1 | Node aggregation |
| Ethernet cables | 2+ | Node to switch |
| 5V supplies or buck per node | 2 | Power |

Search terms (examples): Raspberry Pi 4 2GB, Pi Camera v3, 5 port gigabit switch. Central can be laptop initially; see [platform BOM v1](../platform-bom-v1.mdx).
