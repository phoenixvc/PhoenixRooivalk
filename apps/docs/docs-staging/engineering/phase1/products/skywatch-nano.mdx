# SkyWatch Nano

Entry-level detection node for Phase-1 demos.

## Description and demo goal

Single-node detection; minimal cost; good for first build or proof-of-concept. Camera: prefer OV2640; avoid OV7670 on ESP32. ZAR pricing where useful with USD or local fallback.

## Comparison (1a / 1b / 1c)

| Option | Cost | Performance | Reliability | Best for |
|--------|------|-------------|-------------|----------|
| 1a ultra-cheap | Lowest | Basic | Adequate | First build, learning |
| 1b balanced | Medium | Good | Good | Default demo |
| 1c better | Higher | Best | Best | Demos with headroom |
