# SkyWatch Nano

Entry-level detection node for Phase-1 demos. Can be run as **detection + alarm
only** (no pan/tilt, no actuator): camera → detection → local alarm
(buzzer/strobe) + remote (MQTT/Telegram).

## Description and demo goal

Single-node detection; minimal cost; good for first build or proof-of-concept.
Camera: prefer OV2640; avoid OV7670 on ESP32. ZAR pricing where useful with USD
or local fallback.
