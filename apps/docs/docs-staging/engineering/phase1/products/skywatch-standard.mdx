# SkyWatch Standard

Standard detection node for Phase-1 demos. Step-up from Nano; better compute/camera. Prefer OV2640; avoid OV7670. ZAR where useful with USD or local fallback.

## Comparison (1a / 1b / 1c)

| Option | Cost | Performance | Best for |
|--------|------|-------------|----------|
| 1a | Lowest | Basic | Budget build |
| 1b | Medium | Good | Default demo |
| 1c | Higher | Best | Higher-quality demos |

## BOM (outline)

Compute (e.g. Pi4), camera (OV2640 preferred), power, storage, optional Coral. See [common modules](../../common/index.mdx).

## How-to

Wiring, flash, config, test. Links to [common/compute](../../common/compute.mdx), [avoid list](../avoid-list.mdx).
