# SkyWatch Standard

Standard detection node for Phase-1 demos. Step-up from Nano; better
compute/camera. Can be run as **detection + alarm only** (no pan/tilt, no
actuator) or with optional pan/tilt. Prefer OV2640; avoid OV7670. ZAR where
useful with USD or local fallback.

## Comparison (1a / 1b / 1c)

| Option | Cost   | Performance | Best for             |
| ------ | ------ | ----------- | -------------------- |
| 1a     | Lowest | Basic       | Budget build         |
| 1b     | Medium | Good        | Default demo         |
| 1c     | Higher | Best        | Higher-quality demos |

## BOM (outline)

Compute (e.g. Pi4), camera (OV2640 preferred), power, storage, optional Coral.
See [common modules](../../common/index.mdx).
