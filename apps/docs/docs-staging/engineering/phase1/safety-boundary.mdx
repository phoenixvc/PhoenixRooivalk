# Phase-1 Safety Boundary

Phase 1 is **detection, tracking, alarm, and inert actuation only**. No weaponization.

- **Response Relay:** Relay/MOSFET to LED/lamp/dummy load only; arm/disarm; log activations. No launcher or harmful hardware.
- **Turret:** Pan/tilt + alarm only. No firing hardware.
- **All products:** No building or operating systems intended to shoot down drones; no detailed firing hardware in documentation.

Control-architecture and vehicle-adapter docs, when added, cover detection, tracking, and safe control layer only.
