## 6. Troubleshooting Guide

### Common Issues

| Problem | Likely Cause | Fix |
|---------|--------------|-----|
| "Camera not found" | Ribbon cable loose | Reseat cable, check raspi-config |
| "Model not found" | Wrong path | Check ~/models/ directory |
| Low FPS (<10) | Resolution too high | Use 640x480, not 1080p |
| No detections | Confidence too high | Lower --conf to 0.3 |
| Out of memory | Model too large | Use .onnx not .pt |

### Debug Mode

```bash
# Run with verbose logging
python3 detect.py --debug

# Check camera
libcamera-hello --list-cameras

# Check Python environment
python3 -c "import cv2; print(cv2.__version__)"
python3 -c "from ultralytics import YOLO; print('OK')"
```
