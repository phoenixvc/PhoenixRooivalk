# Kevlar Net Manufacturing Guide

This document covers the manufacturing process for Kevlar capture nets used in
Phoenix Rooivalk products, from raw fiber to finished net.

:::info Phoenix Rooivalk Approach

We manufacture nets in-house from Kevlar fiber stock, giving us control over
quality, specifications, and costs. This guide documents our manufacturing
process and the underlying science.

:::

---

## Part 1: Understanding Kevlar Fiber

### What is Kevlar?

Kevlar is a para-aramid synthetic fiber developed by Stephanie Kwolek at DuPont
in 1965. It's known for:

- **High tensile strength**: 5x stronger than steel by weight
- **Low weight**: Density of 1.44 g/cm³
- **Heat resistance**: Decomposes at ~450°C
- **Chemical resistance**: Resistant to most solvents and chemicals
- **No melting point**: Kevlar decomposes rather than melts

### How Kevlar Fiber is Made (Industry Process)

:::note We Don't Make Kevlar Fiber

Phoenix Rooivalk purchases Kevlar fiber from suppliers (primarily DuPont). The
fiber manufacturing process below is for educational purposes - we convert
purchased fiber into nets.

:::

#### Step 1: Polymerization

```
Monomers:
  1,4-phenylene-diamine + terephthaloyl chloride
                    ↓
           Condensation reaction
                    ↓
         Poly-paraphenylene terephthalamide
                    ↓
              + HCl (byproduct)
```

The reaction occurs at low temperatures in a solution of N-methyl-pyrrolidone
and calcium chloride.

#### Step 2: Fiber Spinning (Wet/Dry-Jet Process)

```
Polymer solution → Spinneret → Air gap → Coagulation bath
        ↓              ↓           ↓            ↓
   Dissolved in    Extrusion    Molecular    Acid removed,
   sulfuric acid   into fiber   alignment    fiber solidifies
```

| Stage                | Temperature | Details                        |
| -------------------- | ----------- | ------------------------------ |
| **Extrusion**        | ~100°C      | Through spinneret capillaries  |
| **Air gap**          | Ambient     | ~1cm, molecular alignment      |
| **Coagulation bath** | 0-4°C       | Cold water, acid extraction    |
| **Washing/drying**   | Variable    | Remove residual acid, dry      |

#### Step 3: Post-Processing

- **Heat treatment**: Improves crystallinity and strength
- **Surface treatment**: Enhances bonding for composite applications
- **Twisting/plying**: Creates multi-filament yarns

### Kevlar Grades

| Grade               | Strength | Modulus | Use Case                 |
| ------------------- | -------- | ------- | ------------------------ |
| **Kevlar 29**       | Standard | Medium  | Body armor, ropes        |
| **Kevlar 49**       | High     | High    | Aerospace, composites    |
| **Kevlar 149**      | Highest  | Highest | Advanced composites      |
| **Kevlar 956**      | Standard | Medium  | Netting, cordage         |

For drone capture nets, we primarily use **Kevlar 29** or **Kevlar 956** -
optimizing for strength-to-weight ratio and cost.

## Part 2: From Fiber to Net

### Our Manufacturing Process

```
Raw Materials         Intermediate           Final Product
─────────────────────────────────────────────────────────────
Kevlar fiber    →    Twisted twine    →    Knotted net
(purchased)          (in-house)            (in-house)
                           ↓
                     Edge binding
                     Corner weights
                     Packaging
```

### Required Equipment

| Equipment               | Purpose                        | Estimated Cost |
| ----------------------- | ------------------------------ | -------------- |
| **Twisting machine**    | Create multi-strand twine      | $2,000-5,000   |
| **Netting loom/shuttle**| Hand or semi-auto net knotting | $500-3,000     |
| **Edge binding machine**| Reinforce net perimeter        | $1,000-2,500   |
| **Cutting station**     | Precision cutting to size      | $200-500       |
| **Heat gun/sealer**     | Prevent fraying                | $50-150        |
| **Tensile tester**      | QC strength testing            | $500-2,000     |

### Step 1: Twine Production

#### 1.1 Fiber Selection

| Property         | Specification               | Test Method        |
| ---------------- | --------------------------- | ------------------ |
| **Denier**       | 1000-1500D                  | Weight per length  |
| **Tensile**      | Min 2,920 MPa               | ASTM D885          |
| **Elongation**   | 2.4-3.6%                    | ASTM D885          |
| **Moisture**     | <0.5%                       | Karl Fischer       |

#### 1.2 Twisting Process

```
Single Kevlar filaments
        ↓
  Combine 3 strands
        ↓
  Twist (S or Z direction)
        ↓
  Urethane coating (optional)
        ↓
  Curing/drying
        ↓
  Spool for storage
```

**Twist specifications:**
- **Twist per inch (TPI)**: 4-8 TPI
- **Direction**: S-twist for final product
- **Tension**: 0.5-1.0 N per strand

### Step 2: Net Construction

#### 2.1 Knotted Net Method (Preferred)

```
Net Construction Pattern:

     ●─────●─────●─────●
     │╲   ╱│╲   ╱│╲   ╱│
     │ ╲ ╱ │ ╲ ╱ │ ╲ ╱ │
     │  ●  │  ●  │  ●  │   ← Knot points
     │ ╱ ╲ │ ╱ ╲ │ ╱ ╲ │
     │╱   ╲│╱   ╲│╱   ╲│
     ●─────●─────●─────●

     Mesh size: {NETS.AIR_TO_AIR_MESH}
```

**Knot types:**
- **Sheet bend**: Standard netting knot
- **Double sheet bend**: Higher security
- **Fisherman's knot**: Maximum strength

#### 2.2 Knotless Net Method (Alternative)

| Pros                     | Cons                      |
| ------------------------ | ------------------------- |
| No weak points at knots  | Requires special loom     |
| Smoother surface         | Higher equipment cost     |
| Faster production        | Less repair flexibility   |

### Step 3: Edge Reinforcement

```
Cross-section of edge binding:

         Net mesh
            │
     ┌──────┼──────┐
     │      │      │
─────●══════●══════●─────  ← Edge rope (5mm braided)
     │      │      │
     └──────┴──────┘
            │
     Double stitching
```

**Edge specifications:**
- **Rope diameter**: 5mm braided Kevlar or nylon
- **Stitch spacing**: 25mm
- **Thread**: Heavy-duty polyester or Kevlar
- **Overlap**: 50mm fold-over

### Step 4: Corner Weights

```
Corner Assembly:

    Net corner
         │
         ▼
    ┌────────┐
    │ D-ring │  ← Stainless steel, 15mm
    └───┬────┘
        │
    ┌───┴───┐
    │Swivel │  ← Ball-bearing, prevents tangling
    └───┬───┘
        │
    ┌───┴───┐
    │Weight │  ← Tungsten, 25g each
    └───────┘
```

| Component    | Material         | Weight  | Supplier Options       |
| ------------ | ---------------- | ------- | ---------------------- |
| **D-ring**   | Stainless steel  | 5g      | McMaster-Carr, Amazon  |
| **Swivel**   | Brass/steel      | 8g      | Fishing supply         |
| **Weight**   | Tungsten/steel   | 25-30g  | Fishing supply, custom |

## Part 4: Quality Control

### Incoming Material Inspection

| Test              | Method                  | Accept Criteria        |
| ----------------- | ----------------------- | ---------------------- |
| **Visual**        | Inspect for damage      | No cuts, fraying       |
| **Tensile**       | Pull test per ASTM D885 | ≥ 2,920 MPa            |
| **Weight/denier** | Precision scale         | Within ±5% of spec     |
| **Moisture**      | Moisture analyzer       | < 0.5%                 |

### In-Process Checks

| Stage         | Check                      | Frequency      |
| ------------- | -------------------------- | -------------- |
| **Twisting**  | TPI, tension               | Every 100m     |
| **Netting**   | Mesh size, knot integrity  | Every 10 knots |
| **Edge**      | Stitch spacing, overlap    | Every edge     |
| **Corner**    | Weight accuracy, swivel    | Every corner   |

### Final Net Testing

```
Test Protocol:

1. Deployment Test
   ├── Drop from 3m height
   ├── Measure opening time
   └── Pass: < 0.5 seconds to full open

2. Strength Test
   ├── Hang 50kg for 1 hour
   ├── Inspect for strand failure
   └── Pass: No breakage

3. Wrap Test
   ├── Deploy onto test frame (simulated drone)
   ├── Measure coverage
   └── Pass: > 80% wrap coverage

4. Cycle Test
   ├── 10 deployment/repack cycles
   ├── Check for tangling
   └── Pass: No deployment failures
```
