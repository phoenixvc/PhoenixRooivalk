## Overview

Major cloud providers offer significant credits and support for startups. Combined, these programs can provide **$500K-$1M+ in cloud resources** for infrastructure, AI/ML, and IoT development.

:::info Total Potential Value

| Provider | Max Credits | Requirements |
|----------|------------|--------------|
| AWS Activate | $100,000 | VC-backed |
| Microsoft Founders Hub | $150,000 | Startups |
| Google for Startups | $350,000 | AI-first |
| NVIDIA Inception | $150,000+ | AI/ML focus |
| **Total** | **$750,000+** | |

:::

## AWS Activate

### Tiers

| Tier | Credits | Requirements |
|------|---------|--------------|
| Founders | $1,000 | Any startup |
| Portfolio | $25,000 | Accelerator/VC |
| Portfolio Plus | $100,000 | Select accelerators |

### How to Apply
1. **Founders Tier**: Self-serve at aws.amazon.com/activate
2. **Portfolio Tier**: Through accelerator or VC partner
3. **Portfolio Plus**: YC, HAX, Techstars alumni

### Relevant Services
- **EC2**: Compute for processing
- **S3**: Storage for drone footage
- **SageMaker**: ML model training
- **IoT Core**: Device management
- **RoboMaker**: Robotics simulation

### Application Timeline
- Founders: Instant approval
- Portfolio: 1-2 weeks via partner
- Portfolio Plus: 2-4 weeks

## Microsoft for Startups Founders Hub

### Credit Tiers

| Stage | Credits | Duration |
|-------|---------|----------|
| Idea Stage | $1,000 | 12 months |
| Building | $5,000 | 12 months |
| Growing | $25,000 | 12 months |
| Scaling | $150,000 | 24 months |

### Pegasus Program
- **Investment**: $250,000 direct investment
- **Credits**: Additional $350,000 Azure
- **Advisors**: Microsoft technical experts
- **Network**: Enterprise customer intros

### Relevant Services
- **Azure VMs**: Compute workloads
- **Azure ML**: Machine learning
- **Azure IoT Hub**: Device connectivity
- **GitHub Enterprise**: Development tools
- **OpenAI API**: GPT-4 access included

### How to Apply
1. Visit foundershub.startups.microsoft.com
2. Complete application (10-15 minutes)
3. Approval: Usually within 48 hours
4. Credits auto-apply to account

## Google for Startups Cloud Program

### Credit Tiers

| Tier | Credits | Requirements |
|------|---------|--------------|
| Standard | $100,000 | Cloud-first startup |
| AI-First | $350,000 | AI as core product |
| Google Accelerator | $500,000+ | Program graduates |

### AI-First Qualification
To qualify for $350K tier:
- AI must be core to product
- Using Google Cloud AI services
- Demonstrate AI roadmap
- Technical architecture review

### Relevant Services
- **Vertex AI**: ML training and deployment
- **Cloud Run**: Serverless compute
- **BigQuery**: Data analytics
- **Cloud Vision**: Image analysis
- **Cloud TPU**: AI accelerators

### Application Process
1. Apply at cloud.google.com/startup
2. Technical review (2-4 weeks)
3. Partner interview if AI-First
4. Credits activated upon approval

## NVIDIA Inception Program

### Benefits

| Benefit | Value |
|---------|-------|
| Cloud Credits | $150,000+ across AWS/Azure/GCP |
| Hardware Discounts | 15-50% off GPUs |
| Technical Support | NVIDIA engineers |
| Go-to-Market | Co-marketing opportunities |

### Hardware Discounts
- Jetson modules: 15-30% off
- DGX systems: Custom pricing
- RTX workstations: 15% off

### How to Qualify
- AI/ML must be core to product
- Working prototype or product
- No cost to join program

### Application
1. Apply at nvidia.com/startups
2. Technical review (1-2 weeks)
3. Acceptance notification
4. Access to benefits portal

## Combined Strategy

### Phase 1: Foundation (Immediate)
1. Apply AWS Activate Founders ($1K)
2. Apply Microsoft Founders Hub ($5K)
3. Apply NVIDIA Inception (hardware discounts)

### Phase 2: Accelerator Boost (With HAX/YC)
1. Upgrade to AWS Portfolio ($25-100K)
2. Upgrade to Microsoft Scaling ($150K)
3. Apply Google AI-First ($350K)

### Phase 3: Ongoing
1. Renew credits before expiration
2. Optimize cloud usage
3. Negotiate custom deals at scale

## Credits Usage Strategy

### High-Value Uses
| Use Case | Provider | Priority |
|----------|----------|----------|
| ML Training | Google Vertex, AWS SageMaker | High |
| Video Storage | AWS S3, Google Cloud Storage | High |
| IoT Backend | AWS IoT Core, Azure IoT | High |
| Simulation | AWS RoboMaker | Medium |
| Analytics | BigQuery, Redshift | Medium |

### Avoid Wasting Credits On
- Always-on VMs (use spot/preemptible)
- Unused storage
- Over-provisioned databases
- Services available free tier
