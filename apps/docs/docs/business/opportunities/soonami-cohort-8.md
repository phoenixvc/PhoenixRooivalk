## About Soonami.io

Soonami.io is a Berlin-based venture capital firm and accelerator founded in 2021.
They describe themselves as "the world's first tokenized venture ecosystem" supporting
a global network of over 10,000 entrepreneurs and 2,000+ projects.

### Key Statistics

| Metric | Value |
|--------|-------|
| **Founded** | 2021 |
| **Headquarters** | Berlin, Germany |
| **Global Network** | 10,000+ entrepreneurs |
| **Projects Supported** | 2,000+ |
| **Previous Participants** | 350+ across cohorts |
| **Countries Represented** | 44+ |
| **Investments Made** | 11 portfolio companies |

### Core Platform: Foundance

Soonami's flagship platform, Foundance, helps startups go from idea to fully on-chain
venture, leveraging Web3 and AI solutions.
