## How to Request Access

### Restricted Materials

For access to restricted materials:

1. **Email**: Contact project administrator
2. **Subject**: "Resource Access Request - [Resource Name]"
3. **Include**:
   - Your name and organization
   - Purpose for access
   - Specific resources needed

### NDA Requirements

Some materials require a signed Non-Disclosure Agreement:

- Investor pitch deck
- Technical specifications
- Competitive analysis
- Source code/SDK

## Download Instructions

### PDF Documents

1. Click the download link
2. Document opens in browser or downloads automatically
3. For restricted files, enter provided password

### Calendar Files (ICS)

1. Download the ICS file
2. Import into your calendar application:
   - **Google Calendar**: Settings > Import & Export
   - **Apple Calendar**: File > Import
   - **Outlook**: File > Open & Export > Import

### Large Files

Files over 100MB are hosted separately. Contact administrator for access
credentials.
