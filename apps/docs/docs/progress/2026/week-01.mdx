## Full Weekly Report

### 1. Pi Drone Detector Application

A new application was developed to demonstrate edge AI capabilities on
resource-constrained hardware, directly supporting the SkySnare product
development.

#### Application Overview

| Component | Technology | Purpose |
| --- | --- | --- |
| **Inference Engine** | TFLite Runtime | Optimized for ARM processors |
| **Detection Model** | YOLOv5n/YOLOv8n | Balance of speed and accuracy |
| **Training Platform** | Azure ML | GPU training with cost control |
| **Target Hardware** | Raspberry Pi 4/5 | Low-cost edge deployment |

#### Classification Taxonomy (10 Classes)

| Class ID | Name | Examples |
| --- | --- | --- |
| 0 | drone | Quadcopters, multirotors, fixed-wing UAVs |
| 1 | bird_small | Sparrows, finches (<40cm wingspan) |
| 2 | bird_large | Eagles, hawks, flocks (>40cm wingspan) |
| 3 | aircraft | Planes, helicopters, gliders |
| 4 | recreational | Kites, balloons, RC planes |
| 5 | sports | Balls, frisbees, projectiles |
| 6 | debris | Plastic bags, paper, leaves |
| 7 | insect | Flies, bees, dragonflies |
| 8 | atmospheric | Rain, snow, lens flare |
| 9 | background | Sky, clouds, nothing detected |

#### Performance Benchmarks

| Configuration | Expected FPS | Detection Range | Notes |
| --- | --- | --- | --- |
| Pi 4 + YOLOv5n (320x320) | 5-8 FPS | 5-30m | Baseline |
| Pi 5 + YOLOv5n (320x320) | 12-18 FPS | 5-30m | 2x faster |
| Pi 4 + Coral USB | 25-35 FPS | 5-30m | 10x speedup |
| Pi 5 + Coral USB | 40-50 FPS | 5-30m | Best performance |
