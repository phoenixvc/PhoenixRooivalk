## Culture & Values

### What We Believe

1. **Speed Wins** — 120ms beats 5 seconds. Always optimize for response time.
2. **Evidence Matters** — Every claim backed by blockchain-verified data.
3. **Dual Revenue** — Consumer funds R&D, enterprise scales the business.
4. **South Africa First** — Build world-class tech from SA, for the world.

### How We Work

- **Remote-first** with co-working in Johannesburg/Cape Town
- **Async communication** — documentation over meetings
- **Ship weekly** — continuous deployment mindset
- **Radical transparency** — open books, open roadmap
